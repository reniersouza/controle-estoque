# 📦 Sistema de Controle de Estoque

Aplicação simples de controle de estoque desenvolvida com HTML, CSS e JavaScript.

O sistema permite cadastrar produtos e visualizar suas quantidades em uma lista.

## 🚀 Funcionalidades

- Adicionar produtos ao estoque
- Informar quantidade de cada produto
- Visualizar lista de produtos cadastrados

## 🛠 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 📚 Objetivo do projeto

Este projeto foi desenvolvido como parte dos meus estudos em Desenvolvimento de Sistemas para praticar lógica de programação e manipulação do DOM com JavaScript.
